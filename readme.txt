字型來源：文泉驛開源矢量中文字體

簡介：
Basic Blind Chess，並不是說此遊戲是用Basic所寫，
而是說此軟体，僅具有基本的AI。
v0.7.2版開始修正大部份長捉的問題，
v0.8.7開始，增加AI搜尋的深度，需使用未來高效能的電腦，
比較跑得動。

Known Issue:從v0.9.4,支援multi processing以後, 
本程式的GUI容易出現Not Responding,
但是在只開本遊戲的情況，等候許久，
大多時候，AI(電腦方)都還是會有回應。
另外本程式v1.1.4用pygame 2.0.0，
會有mixer not initialized的當機問題，
v1.1.5用pygame 1.9.6，有時候音效會關閉，
但至少遊戲不會當掉。

遊戲說明：
按滑鼠鍵(左鍵 或 右鍵皆可)翻開棋子，
而托曳已翻開的我方棋子，可以合法移動或吃子。
規則採用台式暗棋規則。
一開始，隨機決定先手是玩家或電腦，
先手翻開什麼顏色棋，就是用那個顏色棋。
另外，藍色外框表示電腦翻開的棋子。
而右上角，雙劍和盾的圖案，去按它可以投降(重開)。
