Following is Chinese readme:

字型來源：文泉驛開源矢量中文字體

簡介：
Basic Blind Chess，並不是說此遊戲是用Basic所寫，
而是說此軟体，僅具有基本的AI。

遊戲說明：
按滑鼠鍵(左鍵 或 右鍵皆可)翻開棋子，
而托曳已翻開的我方棋子，可以合法移動或吃子。
規則採用台式暗棋規則。
一開始，隨機決定先手是玩家或電腦，
先手翻開什麼顏色棋，就是用那個顏色棋。
另外，藍色外框表示電腦翻開的棋子。
而右上角，雙劍和盾的圖案，去按它可以投降(重開)。

已知主要問題：
